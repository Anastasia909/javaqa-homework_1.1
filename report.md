# Отчёт о тестировании KeyValidator

## Краткое описание

19.03.2020 - 19.03.2020 было проведено функциональное тестирование приложения KeyValidator.

На тестирование затрачено: 1 час 30 минут

В результате тестирования выявлены следующие дефекты:
* https://github.com/Anastasia909/javaqa-homework_1.1/issues/1#issue-584631842


## Описание процесса тестирования

В процессе тестирования использовались следующие артефакты*:
* Инструкция по установке OpenJDK 11
* Руководство использования

В качестве тестовых данных использовались данные Руководства использования KeyValidator:
* При проверке валидных ключей результат - OK 
* При проверке невалидных ключей результат - FAIL

Тестирование производилось в следующем окружении:
* macOS Catalina версия 10.15.3
* openjdk version "11.0.6"
* <другое ПО, при необходимости>
